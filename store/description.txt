AIKeep24-Lite — AI 대화 자동 저장 + 전문 검색

ChatGPT, Claude, Genspark의 대화를 자동으로 저장하고 나중에 검색할 수 있습니다.
LLM/API 없이도 완전히 작동합니다.

【주요 기능】
✅ 자동 저장 — 새 턴 감지 후 30초 idle 시 자동 저장
✅ 한국어 전문 검색 — n-gram 토크나이저 (bigram+trigram)
✅ 영문 검색 지원 — 영어 토큰화 동시 지원
✅ Session Browser — 시간순 세션 탐색 (이번주/이번달/이전)
✅ SNAP — 최근 10턴 클립보드 복사 (컨텍스트 이어붙이기)
✅ 검색 하이라이트 — 매칭 키워드 강조 표시
✅ 단축키 — Cmd+Shift+K (Mac) / Ctrl+Shift+K (Win)

【저장 방식】
• Mode A (기본): 로컬 IndexedDB — 설정 없이 즉시 사용
• Mode B (선택): Cloudflare D1 클라우드 — API Key 설정 후 멀티디바이스 동기화

【지원 플랫폼】
• ChatGPT (chatgpt.com)
• Claude (claude.ai)
• Genspark (genspark.ai)

【개인정보】
모든 데이터는 기본적으로 본인 브라우저에만 저장됩니다.
클라우드 동기화는 선택사항이며 본인 Cloudflare 계정을 사용합니다.
